Avan Utilities plugin helps to have a better website in terms of speed and beauty. The possibilities of this plugin are endless.

=== Description ===
This plugin helps you to have a best wordpress site.
Features of the plugin include speedup wordpress, remove Emoji and dashboard icon, remove jquery migrate, remove woocommerce fragment, remove yoast comment, changing fonts, and more.
Also you can remove classic-themes.min.css  file added since wordpress 6.1 .
It will also be added to the plugin in the near future

== Frequently Asked Questions ==
= How to configure the plugin? =
* Plugin settings are simple and easy to do. Just enter the Avan Utilities settings section from the WordPress dashboard, and after changing the available options, click Save Settings.
= Are your plugin features limited? =
* No, but more features and options will be added in future versions of the plugin.

== Installation ==
= Minimum Requirements =
* WordPress 5.0 or greater
* PHP version 7.2 or greater
* MySQL version 5.0 or greater

= Installation =
1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Avan Utilities > configure any available options > save settings
